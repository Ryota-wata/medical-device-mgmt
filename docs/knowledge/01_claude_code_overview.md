# 第1章：Claude Code概要

## 1.1 Claude Codeとは

### 概要
Claude Codeは、Anthropic社が提供するAIアシスタント「Claude」をコマンドラインから利用できる公式CLIツールです。ターミナル上で自然言語による指示を出すだけで、コードの生成・編集・実行、ファイル操作、Git操作などを自動で行ってくれます。

### 主な特徴

| 特徴 | 説明 |
|------|------|
| **エージェント型AI** | 単なる会話ではなく、実際にファイルを作成・編集・削除できる |
| **コンテキスト理解** | プロジェクト全体のファイル構成を理解した上で作業 |
| **自律的タスク実行** | 複数ステップのタスクを自動で順次実行 |
| **Git連携** | コミット、プッシュ、PR作成まで自動化可能 |
| **プラン機能** | 複雑なタスクは計画を立ててから実行 |

### 従来ツールとの比較

| 項目 | ChatGPT | GitHub Copilot | Claude Code |
|------|---------|----------------|-------------|
| 動作形態 | Webブラウザ | エディタ拡張 | CLI（ターミナル） |
| ファイル操作 | 不可 | 限定的 | **完全対応** |
| 複数ファイル編集 | 不可 | 不可 | **対応** |
| Git操作 | 不可 | 不可 | **完全対応** |
| プロジェクト全体理解 | 限定的 | 限定的 | **対応** |
| 自律的タスク実行 | 不可 | 不可 | **対応** |

### モック駆動開発との相性

Claude Codeは以下の理由でモック駆動開発に最適です：

1. **高速なプロトタイピング**
   - 自然言語で画面イメージを伝えるだけでHTMLやReactコンポーネントを生成
   - 修正指示もテキストで伝えるだけで即座に反映

2. **反復的な改善サイクル**
   - 「ボタンを右に寄せて」「色を変えて」といった細かい調整が容易
   - スクリーンショットを見せてフィードバック可能

3. **設計ドキュメントの自動生成**
   - モックから画面遷移図を自動生成
   - 画面設計書（Excel形式）の自動出力

## 1.2 料金プラン

### Claude Code利用可能プラン

| プラン | 月額 | Claude Code | 特徴 |
|--------|------|-------------|------|
| **Max Plan (5x)** | $100 | 利用可能 | 標準的な利用向け |
| **Max Plan (20x)** | $200 | 利用可能 | ヘビーユーザー向け |
| Pro Plan | $20 | **利用不可** | Webのみ |
| Free Plan | $0 | **利用不可** | Webのみ |

> **注意**: Claude Codeを利用するには**Max Plan以上**の契約が必要です。

### 使用量の確認方法

Claude Code実行中に以下のコマンドで確認できます：

```
/cost
```

表示例：
```
Session cost: $2.45
Total tokens: 150,234 (input: 120,000, output: 30,234)
```

### コスト最適化のTips

1. **CLAUDE.mdの活用**
   - プロジェクトルールを記載することで、毎回の説明が不要に
   - コンテキスト効率が向上

2. **明確な指示**
   - 曖昧な指示は試行錯誤を増やしコスト増
   - 具体的な完成イメージを伝える

3. **スコープを絞る**
   - 「全画面を作って」ではなく「ログイン画面を作って」
   - 1タスク1指示を心がける

4. **サマリー機能の活用**
   - 長時間作業時は `/compact` でコンテキストを圧縮

## 1.3 動作要件

### 対応OS

| OS | バージョン | 対応状況 |
|----|-----------|---------|
| macOS | 10.15以降 | 対応 |
| Windows | 10以降 | 対応（WSL推奨） |
| Linux | Ubuntu 20.04以降等 | 対応 |

### 必要なソフトウェア

| ソフトウェア | 必須バージョン | 用途 |
|-------------|---------------|------|
| **Node.js** | 18.0以上 | Claude Code実行環境 |
| **npm** | 9.0以上 | パッケージ管理 |
| **Git** | 2.0以上 | バージョン管理 |

### 推奨環境

| 項目 | 推奨スペック |
|------|-------------|
| メモリ | 8GB以上 |
| ストレージ | SSD推奨 |
| ネットワーク | 安定したインターネット接続 |
| ターミナル | iTerm2（Mac）/ Windows Terminal（Windows） |
| エディタ | Visual Studio Code（任意） |

### バージョン確認コマンド

```bash
# Node.jsバージョン確認
node -v
# 出力例: v20.10.0

# npmバージョン確認
npm -v
# 出力例: 10.2.3

# Gitバージョン確認
git --version
# 出力例: git version 2.42.0
```
