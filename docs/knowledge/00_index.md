# Claude Codeによるモック駆動開発ナレッジ

## 本書について

本書は、Claude Codeを活用したモック駆動の要件定義・基本設計の進め方をまとめたナレッジ文書です。初めての方でも手順に沿って進められるよう、詳細な説明とプロンプト例を記載しています。

### 対象読者

- Claude Codeを初めて使う方
- モック駆動開発を始めたい方
- 画面設計の効率化を図りたい方
- チームでClaude Codeを活用したい方

### 本書の構成

本書は全12章と付録で構成されています。

## クイックスタート

### 1. 環境構築（第1-2章）

```bash
# Node.jsインストール後
npm install -g @anthropic-ai/claude-code
claude  # 認証フロー開始
```

### 2. プロジェクト初期化（第3章）

```bash
mkdir my-project && cd my-project
claude
```

```
Next.js + TypeScript + Tailwind CSSでプロジェクトを初期化して。
GitHub Pagesで公開する設定も含めて。
```

### 3. CLAUDE.md作成（第4章）

```
このプロジェクト用のCLAUDE.mdを作成して。
UI/UXルール、レイアウトルール、禁止事項を含めて。
```

### 4. 画面モック作成（第7章）

```
ログイン画面のモックを作成して。
ID/パスワード入力、ログインボタンを含む。
レスポンシブ対応（PC/タブレット/スマホ）して。
```

### 5. ドキュメント生成（第9章）

```
全画面の画面遷移図と画面設計書を生成して。
Excel形式で出力して。
```
