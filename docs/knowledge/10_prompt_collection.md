# 第10章：効果的なプロンプト集

## 10.1 環境構築系

### プロジェクト初期化

```
Next.js + TypeScript + Tailwind CSSでプロジェクトを初期化して。

【プロジェクト名】medical-device-mgmt
【追加パッケージ】
- zustand（状態管理）
- zod（バリデーション）
- react-hook-form（フォーム管理）

【設定】
- App Router使用
- src/ディレクトリ不使用
- GitHub Pages用の静的エクスポート設定
```

---

### GitHub連携

```
GitHubリポジトリを作成してプッシュして。

【リポジトリ設定】
- 名前: medical-device-mgmt
- 公開設定: Public
- 説明: 医療機器管理システム画面モック

gh CLIを使用して。
```

---

### GitHub Pages設定

```
GitHub Pagesで自動デプロイする設定を追加して。

【要件】
- mainブランチへのpush時に自動デプロイ
- GitHub Actions使用
- 静的エクスポート（output: 'export'）

.github/workflows/deploy.ymlを作成して。
```

---

### CLAUDE.md作成

```
このプロジェクト用のCLAUDE.mdを作成して。

【含める内容】
1. プロジェクト概要（医療機器管理システムのモック）
2. 技術スタック
3. UI/UXルール
4. レイアウトルール
5. 禁止事項
```

### レスポンシブ対応

```
この画面をPC/タブレット/スマートフォンに対応させて。

【ブレークポイント】
- PC: 1024px以上
- タブレット: 768px〜1023px
- スマホ: 767px以下

【対応方針】
- useResponsiveフックを使用
- isMobile, isTabletで条件分岐
- フォントサイズ、パディング、レイアウトを調整
```
