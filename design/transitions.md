# 画面遷移定義

Figmaのプロトタイプフローを参照し、以下の書式で画面遷移を記載する。

## 書式

```
## 遷移元画面名 → 遷移先画面名
- トリガー: (ボタン名、メニュー項目名など)
- 遷移先: (Next.js ルートパス)
- 備考: (任意)
```

---

## ログイン → メイン画面
- トリガー: ログインボタンクリック
- 遷移先: /main

## メイン画面 → タスク管理（廃棄契約）
- トリガー: サイドメニュー「廃棄契約管理タスク」クリック
- 遷移先: /disposal-task

## メイン画面 → タスク管理（修理）
- トリガー: サイドメニュー「修理タスク」クリック
- 遷移先: /repair-task

## メイン画面 → 保守見積登録
- トリガー: サイドメニュー「保守見積登録」クリック
- 遷移先: /maintenance-quote-registration

## メイン画面 → 見積データBOX
- トリガー: サイドメニュー「見積データBOX」クリック
- 遷移先: /quotation-data-box
